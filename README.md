# Meteor Strike

Trabalho desenvolvido para a disciplina de **Desenvolvimento de Jogos para Smartphones**, parte da pós-graduação em **Programação para Dispositivos Móveis** na **UTFPR**.

## Sobre o Jogo

Meteor Strike é um jogo de ação no estilo arcade desenvolvido na plataforma **Godot 4.6** com **C#**.

Você está a bordo de uma nave espacial tentando retornar à Terra, mas uma chuva de meteoros se aproxima perigosamente. Sua única esperança é o poderoso sistema de mísseis da nave. Destrua os meteoros antes que eles atinjam você e tente chegar são e salvo ao seu destino!
